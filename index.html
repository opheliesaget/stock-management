Comment ça marche ?

Avec cette application mobile, vous pouvez gérer les stocks de votre boutique.

Ajout de nouveau produit : si vous ajoutez une boisson alcoolisée, vous pouvez ajouter le degré d'alcool. L'app vous empéchera d'ajouter un élément si son nom ou sa quantité ne sont pas renseigné, ou si la marge est négative.

Pour modifier ou supprimer un produit : il vous suffit de clicker sur l'élément que vous souhaitez modifier ou le bouton X.

Le stock : un voyant vous prévient lorque votre stock est faible. Il sera vert si le stock est supérieur a 10, jaune entre 1 et 10, rouge si le stock est à zéro.
